# 미야언니 V9 재설계 요약

## 이번 수정 핵심
- 현재 레포 구조는 유지
- 기존 DB/후기/고객 프로필 파일 유지
- 상담 엔진만 자연스러운 흐름 중심으로 재정리

## 반영한 변경
1. `conversation_focus`, `recommendation_mode` 상태 추가
2. 메뉴형 fallback 문구 완화
3. `feature`, `option`, `fit follow-up` 라우팅 추가
4. 대체재 추천(`alternative`)과 코디 추천(`coordi`) 분리
5. 현재 상품과 유사한 다른 상품 요청 시 같은 카테고리 우선 추천
6. 체형 후속 질문(힙/다리짧음/상체/배)에서 이전 사이즈 상담 흐름 유지
7. `티` 표현을 `티셔츠` 카테고리로 더 잘 인식하도록 보강
8. 추천 도입 문구를 상황별로 다르게 생성

## 해결 목표
- "이 슬랙스와 비슷하지만 더 좋은 슬랙스"를 코디 추천으로 오판하지 않기
- "다리도 조금 짧은 편" 같은 후속 질문을 이전 핏 상담의 연장으로 처리하기
- "이 옷 장점이 뭐야", "일자 vs 부츠컷" 같은 질문을 fallback으로 보내지 않기
- 말투를 시스템식 분류 문장보다 매장 상담형에 가깝게 정리하기
